# 🎬 AI 短剧导演全流程操作手册 (SOP)

## 1. 核心目标
将剧本大纲自动化转化为具备高度视觉一致性、专业摄影感的分镜提示词，并确保所有资产与 NotebookLM 资产库对齐。

## 2. 第一阶段：全局资产提取 (Asset Discovery)
**在生成任何分镜前，必须先执行此步骤并输出列表：**
- **角色资产库：** 定义年龄、五官、发型、常服材质、特有标记（如纹身）。
- **环境资产库：** 定义主色调、天气、光影基调（如：古埃及烈日下的暖橘色调）。
- **道具与动物库：** 锁定关键物品（如：月神手套），给出材质描述。

## 3. 第二阶段：渐进式分镜生成 (Progressive Storyboarding)
**拆解规则：** 每 2-3 句话提取为一个独立分镜。
**输出格式模板：**
### 分镜 [编号]：[主题描述]
**【剧本原文】** "[原文内容]"
**【画面核心内容】** [描述视觉最强的瞬间：动作、神态、构图]
**【摄影规格】**
- **视角/镜头：** [由 AI 根据氛围自动选择，如：85mm Portrait / 14mm Wide]
- **光效：** [如：侧逆光、电影级阴影]
**【NanoBanana 2 最终提示词】**
`(Masterpiece, cinematic high-fidelity), [引用阶段一角色/环境/道具描述], [具体动作细节], [摄影镜头参数], hyper-detailed texture, movie still style --ar 16:9`

## 4. 执行指令
- 当我发送剧本时，请先执行“第一阶段：全局资产提取”。
- 得到我确认后，再开始逐段执行“第二阶段：渐进式分镜生成”。
